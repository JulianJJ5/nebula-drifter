> Juego de exploración espacial desarrollado con Canvas y Node.js

## 🎮 Descripción

**Nebula Drifter** es un juego donde controlas una nave espacial en una nebulosa llena de peligros y tesoros. Recolecta fragmentos de éter, esquiva criaturas bioluminiscentes y sobrevive a tormentas de plasma.

## 🚀 Características

- 🎯 Física de inercia realista
- 💎 Sistema de recolección de fragmentos
- 🧬 Criaturas con IA básica
- 🌪️ Tormentas de plasma dinámicas
- 📊 Sistema de puntuación y progresión por zonas
- 🎨 Efectos visuales con partículas

## 🛠️ Tecnologías

- **Frontend**: Canvas API, JavaScript vanilla
- **Backend**: Node.js, Express (planificado)
- **Base de datos**: MongoDB (planificado)
- **Herramientas**: npm, Git

## 📦 Instalación

```bash
# Clonar el repositorio
git clone https://github.com/JulianJJ5/nebula-drifter.git

# Instalar dependencias del cliente
cd nebula-drifter/client
npm install

# Instalar dependencias del servidor
cd ../server
